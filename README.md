# EAR_CANDY
music player

This is music player which is developed in python which is targets people who love music.
It has a simple and attractive UI which is easy to use

#Libraries used:
PYGAME
tkinter
mutagen

Features:

Play
Pause/Resume
Stop
Previous song
Next song
Duration slider
Add and Delete songs
